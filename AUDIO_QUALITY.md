# 🎵 Качество звука в Болтушке

## Технические характеристики

### 🎤 Кодек и битрейт
- **Кодек**: Opus (лучший для голоса)
- **Sample Rate**: 48 kHz (студийное качество)
- **Sample Size**: 24-bit (максимальное качество)
- **Битрейт**: 64-256 kbps (адаптивный)
- **Режим**: Стерео с максимальной полосой пропускания
- **Каналы**: 2 (стерео)
- **Адаптивный битрейт**: Автоматически подстраивается под качество сети

### 🔧 Оптимизации WebRTC

#### Параметры кодека Opus:
- `useinbandfec=1` - Forward Error Correction для устойчивости к потерям пакетов
- `maxplaybackrate=48000` - Максимальная частота воспроизведения
- `stereo=1` - Стерео режим
- `sprop-stereo=1` - Сигнализация стерео
- `cbr=1` - Constant Bitrate для стабильного качества
- `minptime=10` - Минимальное время пакетизации
- `dtx=0` - Отключение DTX для лучшего качества
- `maxbandwidth=20000` - Максимальная полоса пропускания
- `maxframeSize=120` - Максимальный размер кадра
- `maxcapturerate=48000` - Максимальная частота захвата
- `maxptime=120` - Максимальное время пакетизации
- `ptime=20` - Время пакетизации для низкой задержки

#### Приоритеты:
- `priority: high` - Высокий приоритет для аудио треков
- `networkPriority: high` - Высокий сетевой приоритет

### 🎛️ Обработка аудио

#### Профессиональная обработка аудио:
1. **Noise Gate** - Шумовые ворота
   - Threshold: -60 dB
   - Ratio: 20:1
   - Attack: 1ms
   - Release: 100ms

2. **High-pass Filter** - Высокочастотный фильтр
   - Frequency: 80 Hz (убирает низкочастотные шумы)
   - Q: 0.5

3. **3-полосный Эквалайзер**
   - Low Shelf (250 Hz): +2 dB
   - Peaking (1000 Hz): +3 dB (голосовой диапазон)
   - High Shelf (4000 Hz): +1 dB

4. **Low-pass Filter** - Низкочастотный фильтр
   - Frequency: 8000 Hz (убирает высокочастотные шумы)
   - Q: 0.5

5. **Compressor** - Компрессор
   - Threshold: -30 dB
   - Ratio: 4:1
   - Attack: 3ms
   - Release: 100ms

6. **Limiter** - Лимитер (предотвращает клиппинг)
   - Threshold: -3 dB
   - Ratio: 20:1
   - Attack: 1ms
   - Release: 10ms

### 🔊 Настройки микрофона

По умолчанию включены:
- **Echo Cancellation** - Подавление эха
- **Noise Suppression** - Шумоподавление
- **Auto Gain Control** - Автоматическая регулировка усиления
- **High-pass Filter** - Высокочастотный фильтр
- **Typing Noise Detection** - Обнаружение звуков печати
- **DA Echo Cancellation** - Дополнительное подавление эха
- **Noise Reduction** - Дополнительное шумоподавление

### 🌐 Мониторинг сети
- **Адаптивный битрейт**: Автоматически подстраивается под качество соединения
- **Мониторинг потерь пакетов**: Отслеживание в реальном времени
- **Качество сети**: Excellent/Good/Poor с автоматическим переключением битрейта
- **10 STUN серверов**: Максимальная надежность подключения

## 💡 Советы по улучшению качества

### Для лучшего качества:
1. **Увеличьте битрейт до 256 kbps** - максимальное качество
2. **Используйте хорошие наушники** - избегайте эха
3. **Используйте внешний микрофон** - лучше встроенного
4. **Стабильное интернет-соединение** - минимум 500 kbps

### При проблемах:
1. **Снизьте битрейт до 64-96 kbps** - при медленном интернете
2. **Отключите шумоподавление** - если голос звучит "роботически"
3. **Регулируйте чувствительность** - если индикатор не реагирует
4. **Перезапустите настройки** - нажмите "Применить настройки"

## 🎛️ Новые возможности интерфейса

### Настройки звука
Теперь пользователи могут настроить качество звука через удобный интерфейс:

1. **Выбор качества звука**:
   - Низкое (64 kbps) - для экономии трафика
   - Среднее (128 kbps) - оптимальный баланс
   - Высокое (192 kbps) - хорошее качество
   - Максимальное (256 kbps) - студийное качество

2. **Обработка звука**:
   - Включение/выключение шумоподавления
   - Включение/выключение подавления эха

3. **Чувствительность микрофона**:
   - Регулировка от 5 до 50
   - Визуальный слайдер с текущим значением

4. **Мониторинг сети**:
   - Индикатор качества соединения
   - Отображение текущего битрейта
   - Автоматическая адаптация

5. **Техническая информация**:
   - Кодек: Opus
   - Частота дискретизации: 48 kHz
   - Разрядность: 24-bit
   - Каналы: Стерео (2.0)

### Доступ к настройкам
Настройки звука доступны через:
1. Нажмите на свой аватар в нижнем левом углу
2. Выберите "Моя учётная запись"
3. Найдите раздел "Настройки звука"
4. Нажмите "Настроить"

Все настройки сохраняются автоматически и применяются сразу после изменения.

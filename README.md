# 💬 Болтушка

Простой аналог Discord для общения в реальном времени. Создавайте каналы и общайтесь с друзьями!

## ✨ Возможности

- 🔨 Создание текстовых и голосовых каналов
- 💬 Обмен сообщениями в реальном времени
- 🎤 Голосовое общение через WebRTC высокого качества
- 🔇 Управление микрофоном (mute/unmute)
- 🗣️ Индикаторы говорящих пользователей в реальном времени
- ⚙️ Продвинутые настройки звука:
  - Шумоподавление и эхоподавление
  - Регулировка чувствительности микрофона
  - Настройка битрейта (64-256 kbps)
  - Opus кодек с оптимизацией
  - Динамический компрессор и фильтры
  - Стерео режим для максимального качества
- 👥 Список пользователей онлайн
- 🎨 Современный UI в стиле Discord
- 🚀 Простая настройка без регистрации

## 🛠 Технологии

**Backend:**
- Node.js
- Express
- Socket.io

**Frontend:**
- React
- Socket.io-client

## 📦 Установка и запуск

### Предварительные требования
- Node.js (версия 14 или выше)
- npm или yarn

### 1. Установка зависимостей

**Backend:**
```bash
cd backend
npm install
```

**Frontend:**
```bash
cd frontend
npm install
```

### 2. Запуск приложения

**Backend (в первом терминале):**
```bash
cd backend
npm start
```

Сервер запустится на http://localhost:3001

**Frontend (во втором терминале):**
```bash
cd frontend
npm start
```

Приложение откроется в браузере на http://localhost:3000

## 🎮 Как пользоваться

1. При первом запуске введите ваше имя
2. Вы увидите дефолтные текстовый и голосовой каналы
3. Создавайте новые каналы с помощью кнопки "+" (выбирайте тип: текстовый или голосовой)
4. В текстовых каналах: пишите сообщения в чате
5. В голосовых каналах:
   - Нажмите "Подключиться" для входа в голосовой чат
   - Разрешите доступ к микрофону
   - Используйте кнопку 🎤 для mute/unmute
   - Откройте настройки звука (кнопка ⚙️):
     * Включите/выключите шумоподавление
     * Включите/выключите эхоподавление
     * Отрегулируйте чувствительность микрофона (5-50)
     * Выберите качество звука (64-256 kbps)
     * Нажмите "Применить настройки" для сохранения
   - Видите, кто говорит в реальном времени
   - Наслаждайтесь кристально чистым звуком!
6. Справа отображается список пользователей онлайн (в текстовых каналах)

## 🔧 Конфигурация

По умолчанию:
- Backend: http://localhost:3001
- Frontend: http://localhost:3000

Для изменения URL backend отредактируйте константу `BACKEND_URL` в файле `frontend/src/App.js`

REST API Floodilka построен вокруг нескольких ключевых ресурсов. Эта страница — индекс для быстрого перехода к нужному разделу.

## База

- **Base URL:** `https://floodilka.com/api/v1`
- **Формат:** JSON в теле запроса и ответа
- **Аутентификация:** `Authorization: Bot <token>` или `Bearer <access_token>` — см. [Аутентификация](/introduction/authentication/)
- **Лимиты:** глобальный + бакет, 429 с `Retry-After` — см. [Лимиты запросов](/topics/rate-limits/)
- **Ошибки:** `{ "code": "...", "message": "..." }` — см. [Коды ошибок](/topics/error-codes/)

## Snowflake-идентификаторы

Все ID ресурсов — 64-битные snowflake. Никогда не парсите их как число в JavaScript (потеря точности) — используйте `BigInt` или строку. Подробнее: [Snowflake-идентификаторы](/topics/snowflakes/).
